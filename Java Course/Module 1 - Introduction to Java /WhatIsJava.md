## What is Java?
Java is an object-oriented, class-based, high-level programming language. It was created by James Gosling at Sun Microsystems, which is now a part of Oracle Corporation, and it was made available in 1995. The "Write Once, Run Anywhere" (WORA) concept guided the language's development. Platform independence, which enables the same Java application to operate on several systems without changes, is a fundamental characteristic of Java that is highlighted by this principle.

Java is designed to be equally powerful and simple to use. It takes its syntax from both C and C++, but it does away with some of the low-level programming intricacies of C++, such as multiple inheritance and explicit memory management. In contrast to these two languages, Java contains a garbage collector that automatically cleans the application memory, so you don't need to do it yourself. Java has gained popularity among developers all around the world because of its strength, security, and ease of use. It supports a variety of programming approaches, such as functional programming, multithreaded and concurrent programming, and generic programming.

### A Sample of Java

Let's code the classic "Hello, World!" program, a friendly greeting from your computer. :computer:

    public class HelloWorld {
      public static void main(String[] args) {
          System.out.println("Hello, World!");
      }
    }

If it appears a little cryptic now, don't worry. You'll easily get the hang of it.

All that this program does is print "Hello, World!" to the terminal. However, a lot is happening here:

  - `public class HelloWorld` - All of the code you write in Java will be contained in classes. We are informing Java that a new public class—a sort of blueprint—will be created and given the name HelloWorld. There must be at least one class in every Java program.

  - `public static void main(String[] args)` - This is the heart of our program, where the execution starts.

  - `System.out.println("Hello, World!");` - These are the opening words of our show! This command provides immediate feedback by telling Java to use the System.out.println() function to print "Hello, World!" to the console.

Keep in mind that there are several kinds of brackets used in Java code, and you must always close any open brackets.

Your very first Java application is now complete! Though it's a little step, it's the start of an amazing adventure into Java programming. :slightly_smiling_face:
